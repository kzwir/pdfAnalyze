# pdfAnalyze
Automatyczny pipeline do przetwarzania dokumentów PDF, ekstrakcji tabel oraz generowania raportów Microsoft Word (.docx).  Projekt obsługuje przetwarzanie pojedynczych plików oraz całych katalogów PDF przy użyciu wielu silników ekstrakcji, przetwarzania równoległego, mechanizmów fallback oraz systemu metryk jakości.
